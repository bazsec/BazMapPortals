# BazMapPortals

![WoW](https://img.shields.io/badge/WoW-12.0_Midnight-blue) ![License](https://img.shields.io/badge/License-GPL_v2-green) ![Version](https://img.shields.io/github/v/tag/bazsec/BazMapPortals?label=Version&color=orange)

Adds clickable portal and teleport spell icons directly to the world map for mages.

BazMapPortals places pin icons on the world map at each destination a mage can teleport or portal to. Click a pin to cast the spell directly from the map. Supports all current teleport and portal destinations including Midnight's Silvermoon City for both factions.

***

## Features

*   **Map pins** at every teleport and portal destination
*   **Click to cast** - cast teleport or portal spells directly from the world map
*   **All destinations** - includes every mage teleport and portal in the game
*   **Midnight updated** - Silvermoon City (Midnight) teleport and portal for all factions
*   **Map overrides** for Silvermoon City, Quel'Thalas, and Midnight Eastern Kingdoms zones
*   **Profile support** via BazCore

***

## Slash Commands

| Command | Description |
| --- | --- |
| `/mp` | Open settings panel |
| `/mp set <city>` | Save a custom coord override at your current map position |
| `/mp clear <city\|all>` | Clear a single override or all of them |
| `/mp dump` | Print current overrides |
| `/mp info` | Show diagnostic info about the current map |
| `/mapportals` | Alias for `/mp` |

***

## Compatibility

*   **WoW Version:** Retail 12.0 (Midnight)
*   **Class:** Mage only (pins only appear for mage characters)
