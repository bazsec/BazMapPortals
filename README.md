# BazMapPortals

Adds clickable portal and teleport spell icons directly to the world map for mages.

Part of the **Baz Suite** of World of Warcraft addons, powered by BazCore.

## Features

- Clickable spell icons on the world map for all mage teleports and portals
- Left-click to teleport, right-click to open a portal
- 25+ destinations across all expansions
- Automatic coordinate projection across map zoom levels
- Faction-aware destination filtering
- Custom coordinate overrides via slash commands
- Map tracking dropdown toggle

## Commands

- `/mp` - Open settings
- `/mp set <city>` - Save current position for a city
- `/mp clear <city|all>` - Clear saved coordinates
- `/mp dump` - Print all saved portal locations
- `/mp info` - Show current map ID and player coordinates

## Requirements

- BazCore (required dependency)
